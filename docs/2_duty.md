# 2_实验室值日

*碍于财力不足，本课题组暂时无力聘请实验室管家处理实验室各项事务*，包括但不限于**实验室卫生**、**实验设备维护**、**药品及耗材采购**、**实验废物处理**、**常规仪器上机等**。当你正式进组时，会统一进行实验室值日工作的排班，该排班计划**持续至下一届研二新生进组**。原则上课题组所有成员均有责任和义务深度参与课题组各项事务的值日与管理中来，但**研二同学将按照惯例承担主要的值日工作**。对于具体的值日细节，你可以查看届时具体的排班表了解。下面将简要介绍每项值日工作的具体SOP，以帮助你尽快上手工作，并了解实验室的日常运作方式。

## 2.1_实验室卫生

实验室（*此处特指实验16楼10层*）每天处理一次常规实验垃圾，除周五外，**每天安排的值日生应在傍晚18:00左右清理一次各实验室的垃圾桶**：若垃圾桶满于1/2，需将垃圾打包丢至卫生间处的大垃圾桶，并重新套上垃圾袋（垃圾袋一般存放在08进门的抽屉下）；**若房间内有明显垃圾，需进行清扫与整理**。**每周五下午17:00为实验室大扫除**，各值日生需对负责的实验室进行一次深度的整理与清洁，具体包括：整理实验台并摆放整齐器具、擦拭桌面与设备灰尘、扫地、拖地、无人认领实验材料的清理、清理垃圾桶并重新套上垃圾袋。除节假日外，工作日与周末均需正常值日；**无法值日的则无需请假，私下调节值日工作拜托同门代班即可**。每日值日生完成工作后均需在对应房间的值日表上签字，否则视为缺勤。

## 2.2_实验设备维护

为了确保实验室正常运转，**本课题组为所有实验设备安排了对应的负责人**。对于各个课题的关键设备，一般由对应的博士生和高年级硕士生负责。除此之外，对于一些常规、常用的实验设备，则需要值日生定期维护。为了防止小小炉使用过度产生积碳，每周一早上，值日生需持特定的管刷对小小炉石英管进行清理并进行空烧；同样地，各个房间的空调与冰箱需要值日生每隔半年清洗一次滤网与除冰。

## 2.3_药品及耗材采购

对于一些通用试剂、气体等，由轮换的值日生负责进行采购与入库工作。具体来说，由值日生在相应平台上下单（地址可直接填对应实验室，一般溶剂可填1008，物流会直接送到实验室），到货后及时贴上二维码，在HSE系统上完成入库；去离子水桶放置在08窗台，值日生每周去水站打一次水（*需支付水票*）；液氮桶放置在06，值日生在实验12楼旁进行液氮补充；而一些常用实验耗材（*例如卷纸等*）以及某些专用实验耗材（*如电池壳等*），则由值日生进行统一采买与相应的报销工作。

## 2.4_实验废物处理

实验废物包括固废（*空试剂瓶/桶、废手套、碎玻璃等*）、废液。值日生每周四应统一清理一次实验废物（*无论废物量多或少*），用推车转移至回收站统一处理，然后重新为固废箱贴上标签、拿回新的废液桶。

## 2.5_常规仪器上机 {: #2.5 }

对于一些常用仪器（*如SEM、XRD、Raman等*），本课题组每年会由专人进行该仪器的上机培训，承担组内部分测样工作。组内同学若有测试需求，可直接联系相应负责同学安排测试，但**需提供样品的详细信息与测试参数，同时确保测试时在场**，并及时缴费。
