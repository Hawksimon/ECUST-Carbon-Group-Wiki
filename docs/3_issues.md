# 3_日常事务

在你的科研过程中，势必会处理一些固定的事务，包括**个人实验材料的采买**、**财务报销**、**仪器预约**等。具体的细节可能会随着时间变动，你进组后可以从你的师兄师姐那里了解当下最新的SOP。在本节本Wiki仅对撰写的当下的这些事务的处理方式做出概括，为你扫盲。

## 3.1_个人实验材料的采买

上面提到，一些常规溶剂和常用耗材会安排值日生统一购买；但每个人的体系都有独特性，**对于那些仅供你自己使用的实验材料，需要你自己从特定渠道购买**。由于课题组历届课题体系差异不大，因此许多试剂课题组或许都有库存，在购买前你必须先登录学校的HSE平台（*需要提前联系导师将你的个人信息加入课题组*），在平台上查找你需要的药品库存信息，**若无库存再选择购买**。对于绝大部分化学试剂，你可以从学校的虫洞平台上下单（*需要提前联系导师将你的个人信息加入课题组*）：

[虫洞平台](https://clcg.ecust.edu.cn/)

平台采用华理统一身份认证登录，登录后建议你通过CAS号精确查找你需要的化学品，你可能注意到不同商家的价格迥异，一般建议你购买**泰坦**、**阿拉丁**、**毕得**、**安耐吉**、**乐研**、**麦克林**等平价药品，对绝大多数的实验而言足够了。注意尽量选择自有品牌或制造商的现货药品。试剂购买量也不易过大，若你只是尝试一个新体系，购买一次使用量的1-3倍即可（*例如你一次实验预计使用10g，那么应购买10g包装药品，25g完全足够了*）。将你选择的药品加入购物车后提交订单，选择送货地址（*一般为实验16楼1008*）、导师工号（*一般为马老师，09144*）、以及届时使用中的经费卡号，你就可以顺利下单了！对于常规现货药品，你总能在5个工作日内收到。

!!! info
	对于易制毒易制爆药品，需要额外提交使用台账（HSE平台截图）

特别地，对于某些特定的实验材料（*例如石英砂、分子筛*），**在淘宝上下单则更为便宜**，但缺点是需要你自己开票报销。在下单界面选择开票-普通发票（电子），选择抬头类型为企业，输入发票抬头为华东理工大学，纳税人识别号为12100000425010757A，即可开具正常的发票。若下单界面没有开票选项，你也可以联系客服开票。开票完成后，你会收到一张盖有公章的电子发票。

## 3.2_财务报销

当你自费垫付了购买实验材料的价钱，你可以在收到发票后通过流程开始财务报销，很快你垫付的金额会打到你的学校农行卡中。你可以登录**华理财务处**官网开始报销：

[财务处](https://cwc.ecust.edu.cn/WFManager/home2.jsp;jsessionid=E9565EDC703AEC9E32C2F484EE8C3CB3;jsessionid=6A0DE79E08A4581C315C47FBBF970271)

进入官网后选择**网上预约报账**，点击**申请报销单**，填写届时使用的经费卡号与其他基本信息，通过指引流程完成报销单的创建（*需要导师审核通过*）。报销单创建完成后，对于一般的实体的实验材料，还需登录**实装处**官网进行审核：

[实装处](https://wzsh.ecust.edu.cn/sso/index.do)

进入实装处后选择**发票审核（无请款）**，在跳转页面中填写你预约的报销单号与发票各项信息，即可提交审核。审核完成后，你需要打印下你预约的报销单与实装处审核单（*对于无需实装处审核的发票则不需要*），以及打印下你的所有发票，签上你自己的名字并找导师签名；合并上述材料前往**研究生楼一楼大厅**，有一台**自助投递机**，旁边通常会有一个放有大自封袋的箱子，你可以拿一个自封袋将你所有的材料装起来，让机器扫描你的报销单右上角的二维码，你就可以将袋子投入机器。**之后就可以等待报销通过了**（*不过你需要注意你的手机短信，如果报销单存在问题，财务处老师会通过短信联系你*）！

## 3.3_仪器预约

在你的科研工作中，你所用到的大部分仪器都通过预约获取使用资格。除去一些高端表征，大部分常见设备，主要依托**以下平台**进行（*需要加入课题组*）：

* 课题组自有仪器设备

* [华理分析测试中心](https://yqgx.ecust.edu.cn/)

* [化工学院985测试平台](http://172.21.41.26/index.aspx)

* [材料学院测试平台](clpt.ecust.edu.cn)

* 诺奖中心

* [科学指南针](https://www.shiyanjia.com/)

对于大部分合成工作与性能测试，课题组均能够提供相关的设备。除实验16楼1007的**水热合成用鼓风干燥箱**的使用通过房间内本子登记外（*1001手套箱的使用也需登记，但无需预约*），其他绝大部分仪器设备均依托**微信预约小程序或共享文档**预约，你可以询问师兄师姐某仪器的具体预约方式。仪器大概包括：**各种炉子**、**氮气吸脱附测试仪**（BET）、**热重分析仪**、**电化学工作站**（1001）、**球磨机**。其他未提到的设备一般由于使用人数较少可不用预约。

除此之外，如之前 [2.5_常规仪器上机](2_duty.md#2.5) 所述，课题组安排了专人负责**SEM**（*实验16楼1楼*）、**XRD**（*实验16楼4楼*）、**Raman**（*分测中心3楼*）的上机测试。从这里开始，进行的课题组外测试均需正常缴纳测试费用，而校内平台的测试经费向导师提出申请，批准后划入校园卡的经费账户中使用；如需进行测试，**须得到导师批准**。如你需要在校内的测试平台进行测试，可在上方链接的官网中查询详细仪器信息、送样条件与收费标准。**原则上禁止擅自测试，禁止铺张浪费**。

若校内测试平台满足不了你的测试需求，你可以在网上找寻满足要求的测试机构。本课题组和科学指南针有合作与价格优惠，若送样，可在平台上下单后打印送样单，将送样单与测试样品装入实验16楼10层玻璃房外透明盒子中，科学指南针会有专人前来收取。
