# 4_实验室规章

为了课题组所有同学实验顺利开展、维护课题组和谐，在这里再次重申，**你应该在进组前了解并将遵守以下规章：**

1. 注重实验安全，实验时做好自己的安全保障，并及时提醒他人实验危害与做好防护，**在不损害自己与他人身体健康的前提下开展实验**；

2. 每次做完实验，一定要确保放回试剂、**清理实验垃圾**（包括称量纸、药匙、甚至测试完成的旧电池）、**用抹布擦拭实验台保证整洁**；

3. 各个实验室的洗瓶（或卷纸、滴管等耗材），**用完了请及时补充**，禁止跨房间使用，**不要永远让其他同学为你服务**；

4. 若常规溶剂或实验耗材即将耗尽，**请及时联系值日生购买**，否则会耽误所有同学的实验；

5. 若不小心破坏其他同学的实验，**应及时在大群或私下主动联系当事同学**，而不是等对方找上门；

6. 每位同学都有承担组内额外工作的责任；

7. 端正实验态度，**爱护设备与仪器**，至少不要让你的随意产生的后果影响到其他同学的正常实验；
